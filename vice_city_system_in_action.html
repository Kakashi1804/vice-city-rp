System in action
Live example — the drug supply economy · Four phases · Automatic consequences
01Import phase
The Cartel receives a shipment. This is where the economy starts — and where the first control point sits.
Who actsCartel leadership initiates the import. Dock control determines what happens next.
Admin roleRead the dock control log. Apply the probability table. No manual storytelling needed — the system tells you what fires.
IF Cartel controls docks → shipment arrives intact // baseline, no event
IF Kings control docks → 40% theft probability per run
IF Smugglers control → 70% theft probability // they have motive + access
IF VCPD controls docks → shipment seized → Cartel-VCPD war arc triggers
Key mechanicThe Cartel sets the global drug price at the point of import. Every dollar the Kings earn this week is calculated against that rate — before they even know it changed.
02Distribution phase
Product moves from the Cartel to the street. Prices fluctuate. The Kings feel it first.
Who actsOvertown Kings purchase supply from the Cartel at the current rate and distribute to their street corners.
Cartel sets price
→
Kings buy at that rate
→
Kings sell on streets
→
Margin = street price − Cartel rate
Price spikeCartel raises price 20% → Kings' margin compresses → Kings have three options: absorb the loss, raise street prices (risk territory), or cut the Cartel out entirely (triggers conflict event).
Price dropCartel floods supply to undercut Kings' independent channels → Kings lose street customers → turf value collapses → territory war becomes economically rational.
This is not flavor. These are mechanical pressure points admins can pull at any time to engineer conflict without scripting a single scene.
03Disruption phase
Any faction can intercept the supply chain. This is where alliances break and betrayals happen.
Who can disruptSmugglers (intercept mid-transport), Kings (rob a Cartel handoff), VCPD (seize at point of sale), rival factions (plant false intel about shipment routes).
Smuggler intercept
TriggerSmugglers accept a paid contract from any faction to intercept a Cartel shipment.
MechanicIf successful: intercepted product enters a black market — anyone can bid on it. Cartel loses product and revenue. Smugglers gain leverage.
RiskIf caught: Cartel declares open war on Smugglers. Weapons supply to ALL factions becomes hostile territory.
VCPD seizure
TriggerVCPD activity level crosses a threshold — triggered by how visible the drug economy has been that week.
MechanicSeizure removes product from the server economy for 48h. Scarcity event fires automatically — street price doubles.
RiskIf Syndicate has bribed the right VCPD commander, the seizure is staged — product is actually routed to the Syndicate's laundering operation.
04Law enforcement response
VCPD response is not random — it is calculated from a live activity score that factions generate by playing normally.
How it worksEvery faction action generates visibility points. When total server visibility crosses a threshold, the VCPD response tier escalates automatically.
VISIBILITY_SCORE += drug_sale × 1
VISIBILITY_SCORE += public_conflict × 3
VISIBILITY_SCORE += failed_transport × 2
VISIBILITY_SCORE -= bribe_paid × 4 // Syndicate mechanic

score < 30 → VCPD passive // patrol only
score 30–60 → VCPD active // targeted raids
score 60+ → VCPD crackdown // server-wide enforcement event
Syndicate leverThe Syndicate can pay to suppress the visibility score — making VCPD look the other way. This is their most valuable service and why every faction eventually needs them.
★Outcomes — what the system produces
Player choices determine which of these fires. Admins select from the appropriate branch. Nothing is predetermined.
Economy outcome

Economy outcome
Surplus
Cartel floods market. Prices crash. Kings' territory loses value. New alliances form around volume deals.

New alliance

None of these outcomes required a single admin to write a story. The system ran the economy. Players made choices. The consequences fired automatically.
Admin control flow
Three control levers · Automatic triggers · Manual escalations · Outcome selection
The three things admins actually control
Lever 1
Event triggers
Manual or automatic. Admins set conditions. Player behavior fires the branch. Admins never need to write what happens — they select from what the system produces.
Lever 2
Faction influence
Territory control, loyalty logs, activity scores. Updated every 72h. Feeds directly into trigger calculations and finale outcome scores.
Lever 3
Outcome selection
At resolution points, admins choose from 2–3 pre-written branches based on measurable player activity. No improvisation. No favorites. The data decides.
Conditional trigger examples
IF 3+ factions compete for same territory node → trigger TERRITORY WAR event
Admin action
Open 48h declaration window. Each faction must declare: fight, negotiate, or retreat.
If no declaration
Force-assign neutral status. Territory goes to auction — highest activity wins.
Outcome
Territory winner gets a permanent node. Losers lose a season resource. Logged for finale score.
IF Cartel loses its supply shipment → trigger SCARCITY ECONOMY
Admin action
Post street price doubled in faction channels. Open emergency supply mission for any faction.
If Smugglers fill gap
Smugglers gain Cartel-level leverage for 1 week. Cartel must negotiate or retaliate.
Outcome
Economy resets at end of scarcity window. Power ranking updated. Visibility score spiked — VCPD responds.
IF VCPD visibility score exceeds 60 → trigger CRACKDOWN event
Admin action
Announce server-wide crackdown. VCPD gets expanded powers for 72h.
If Syndicate bribes
Crackdown is staged — visible to public, but product routes around it. Syndicate gains political capital.
Outcome
Crackdown ends. Visibility score resets. Factions that got caught lose resources. Clean factions gain standing.
IF any faction drops below 5 active members → trigger NEW BLOOD emergency event
Admin action
Announce open recruitment war. Other factions can poach members. Struggling faction fights to survive.
If faction survives
Gains a "survivor" buff — one free trigger immunity for rest of season.
If faction falls
Disbanded. Members absorbed into other factions. Territory redistributed. Season narrative adjusts.
The 48-hour rule — why it works
What it isEvery open trigger gives players a 48-hour window to act before admin escalation fires automatically. Players who engage shape the outcome. Players who miss the window lose narrative control.
Why it worksIt creates urgency without manufactured pressure. The deadline is structural, not emotional. Players feel the stakes because the clock is real.
Admin ruleNever extend the window. The moment admins start bending the 48h rule, players learn the deadlines aren't real — and engagement drops immediately.
Finale score calculation
What it isEvery season finale outcome is calculated from measurable activity data accumulated across the whole season. Not vibes. Not admin favorites. Numbers.
SEASON_SCORE = (territory_held × 2) + (deals_completed × 1.5)
+ (events_won × 3) + (loyalty_unboken × 2)
- (betrayals_caught × 2) - (wars_lost × 1)

Highest score faction → gets finale advantage
Score is calculated privately // never reveal the formula
Admin notePlayers should feel like their choices mattered — not like they gamed a spreadsheet. Announce only the result. Never the math.

Daily login hooks — why players come back tomorrow
24h jobOne repeatable faction job per rank, per day. 15–20 minutes. Earns rank XP. Always available, always expiring.
6h intelRandom tip drops in faction channel. First player to act gets the bonus. Creates FOMO without manufactured drama.
7-day streak7 consecutive active days = 1.5× XP multiplier. 30-day streak = faction-wide public reward announced to the server.
Loyalty clockMiss 5 days without notifying leadership = automatic one-tier demotion. The consequence is structural, not personal.
Weekly dealEvery faction has one open deal slot per week. Any other faction can request. Accepting earns both sides XP. Refusing is logged.
Herald postWeekly lore recap names specific player decisions. Players check it to see if they were mentioned. They always come back for that.
